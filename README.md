# Gesture Controlled Synth

A browser-based instrument you play with your keyboard and one hand. Built with MediaPipe, Tone.js, and my novice coding skills.

## How to use

1. Open `index.html` with Live Server in VS Code
2. Click anywhere on the page to activate audio
3. Allow webcam access when prompted

## Controls

**Keyboard** — play chords (keys 1-8, [more coming soon!])

**Hand (right hand in front of webcam)**
- Move **up/down** to control volume — hand out of frame fades to silence
- **Tilt** index finger up or down to change the tremolo speed (snaps to the bpm)
- Move **left/right** to sweep the filter — left is low tones, right is high

**BPM slider** — set the tempo for tremolo and metronome 

**Metronome button** — toggle a click track on/off

## Setup

Just a browser and a webcam. No installs needed beyond VS Code with the Live Server extension.

Works best with wired headphones.
